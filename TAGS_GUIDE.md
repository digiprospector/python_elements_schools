# 题目分类说明

## 加法题目分类

### 数字位数
- **包含一位数（第一个数）**: 第一个加数是1-9
- **包含两位数（第一个数）**: 第一个加数是10-99
- **包含一位数（第二个数）**: 第二个加数是1-9
- **包含两位数（第二个数）**: 第二个加数是10-99

### 进位情况
- **需要进位**: 个位相加 ≥ 10，例如 8+5=13
- **不进位**: 个位相加 < 10，例如 3+4=7

### 结果范围
- **结果大于99**: 答案 > 99，例如 50+60=110
- **结果大于50**: 答案 > 50，例如 30+25=55
- **结果小于等于20**: 答案 ≤ 20，例如 5+8=13

## 减法题目分类

### 数字位数
- **被减数是一位数**: 被减数是1-9
- **被减数是两位数**: 被减数是10-99
- **减数是一位数**: 减数是1-9
- **减数是两位数**: 减数是10-99

### 退位情况
- **需要退位**: 被减数个位 < 减数个位，例如 13-5=8
- **不退位**: 被减数个位 ≥ 减数个位，例如 15-3=12

### 数值范围
- **被减数大于50**: 被减数 > 50
- **结果为0**: 答案 = 0，例如 5-5=0
- **结果小于10**: 答案 < 10，例如 12-5=7
- **结果大于等于50**: 答案 ≥ 50，例如 80-20=60

## 使用建议

### 针对不同学习阶段

**初级阶段（刚开始学习）**
- 加法：选择"不进位" + "结果小于等于20"
- 减法：选择"不退位" + "结果小于10"

**中级阶段（掌握基础）**
- 加法：选择"需要进位" + "结果小于等于20"
- 减法：选择"需要退位" + "结果小于10"

**高级阶段（熟练掌握）**
- 加法：选择"结果大于50"或"结果大于99"
- 减法：选择"被减数大于50"

**专项练习**
- 进位练习：只选择"需要进位"
- 退位练习：只选择"需要退位"
- 大数练习：选择"包含两位数"相关选项

## 组合使用

可以同时选择多个条件，系统会找出同时满足所有条件的题目。

例如：
- 选择"需要进位" + "结果小于等于20" = 简单的进位题
- 选择"需要退位" + "被减数是两位数" = 两位数退位减法
- 选择"包含一位数（第一个数）" + "包含两位数（第二个数）" = 一位数加两位数

如果选择的条件过于严格，可能找不到足够的题目，系统会提示调整筛选条件。
